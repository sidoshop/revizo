Comment l'État peut-il intervenir dans l'économie ?
Terminale STMG · 1 page · Encoder · Comprendre · Récupérer
"Pourquoi l'État intervient-il, avec quels outils, et quelles limites ?"
ÉÉ
Le socle · ★★★★★
L'ÉTAT DANS L'ÉCONOMIE — 3 fonctions
L'État remplit 3 fonctions : allocation des ressources (fournir les biens publics), redistribution (réduire les inégalités), stabilisation (maintenir croissance et emploi).
Allocation des ressources Redistribution Stabilisation
Chaîne logique du thème 6
Le marché a des défaillances → L'État intervient → Via le budget (budgétaire) → Via la BCE (monétaire) → Via la redistribution → Mais dette + inflation = limites
Outil n°1 · L'État ★★★★★
Politique budgétaire
L'État utilise son budget (dépenses + impôts) pour agir sur l'économie. Deux approches : la relance par la demande (augmenter les dépenses ou baisser les impôts) et la politique d'offre (baisser les charges des entreprises, favoriser l'innovation).
Relance (demande) Politique d'offre
Ex. : plan de relance post-Covid de 100 Mds EUR (2020)
Comment l'État finance ★★★★☆
Dépenses, recettes, déficit, dette
Les dépenses publiques sont financées par les prélèvements obligatoires (impôts + cotisations, ~45% du PIB). Quand les dépenses dépassent les recettes = déficit public (flux annuel). L'accumulation des déficits = dette publique (stock).
Prélèvements obligatoires Déficit public Dette publique
Outil n°2 · La BCE ★★☆☆☆
Politique monétaire
La BCE agit via les taux directeurs. Objectif : inflation ~2%. Taux bas → crédit facile → relance de l'activité. Taux élevés → crédit cher → frein à l'inflation. Comprendre la politique monétaire nécessite de connaître la création monétaire et le taux d'intérêt.
Taux bas → relance Taux élevés → frein BCE Création monétaire Taux d'intérêt
Ex. : BCE remonte les taux en 2022-2023 contre l'inflation post-Covid
Outil n°3 · Solidarité ★★★☆☆
Redistribution
L'État transfère des revenus pour réduire les inégalités. Deux formes : la redistribution horizontale (protéger contre les risques : maladie, chômage, vieillesse) et la redistribution verticale (réduire les écarts de revenus entre riches et pauvres).
Horizontale Verticale
Ex. : salarié malade indemnisé (horizontal) ; cadre paye plus d'impôts (vertical)
Pourquoi l'État intervient ★★☆☆☆
Les défaillances du marché
Quand le marché échoue seul, l'État intervient. 4 cas : l'asymétrie d'information (un acteur sait plus que l'autre), les externalités (effets sur des tiers sans compensation), les biens publics (non rivaux et non excluables), la concurrence imparfaite (monopoles).
Asymétrie d'info Externalités Biens publics Concurrence imparfaite

Confusions classiques au Bac
⚠️ Budgétaire = l'État (budget) ≠ Monétaire = la BCE (taux)
⚠️ Déficit = flux annuel (cette année) ≠ Dette = stock (toutes les années)
⚠️ Bien public = éclairage public (éco) ≠ Service public = école, hôpital
⚠️ Politique de demande = relancer la conso ≠ Politique d'offre = aider les entreprises
Trucs pour retenir
ARS = les 3 fonctions : Allocation · Redistribution · Stabilisation
Déficit → Dette : le déficit est le robinet, la dette est la baignoire
BCE = taux : la BCE ne touche qu'au taux (directeur). L'État touche au budget
Bien public = 2 "non" : non rival + non excluable
Relance = + ou - : + dépenses OU - impôts → + consommation → + PIB
Externalité = sans payer : effet sur un tiers SANS compensation
