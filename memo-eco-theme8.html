Mémo · Économie · Thème 8
Comment organiser le commerce international ?
Terminale STMG · 1 page · Encoder · Comprendre · Récupérer
"Pourquoi les pays échangent-ils, et comment organiser ces échanges ?"
CI
Le contexte ★★★★☆
La mondialisation
La mondialisation est l'ouverture des économies sur un marché planétaire. Les échanges sont portés par les FMN (firmes multinationales qui produisent dans plusieurs pays), la division internationale du travail (chaque pays se spécialise) et l'internationalisation croissante des échanges de biens, services et capitaux.
Mondialisation Internationalisation FMN DIT
Le grand débat ★★★★★
Libre-échange ou protectionnisme ?
Deux visions s'affrontent : le libre-échange (supprimer les barrières — les prix baissent, chacun se spécialise) contre le protectionnisme (limiter les importations — protéger l'emploi national et les industries fragiles). C'est LE sujet d'argumentation au Bac.
Libre-échange Protectionnisme
Chaîne logique
Économies s'ouvrent → Libre-échange vs Protectionnisme → OMC régule → Balance commerciale + IDE → Croissance ou inégalités ?
Pour l'ouverture ★★★★★
Le libre-échange
Pourquoi ouvrir les frontières ? Les économies d'échelle (produire plus = coûter moins cher par unité), la spécialisation (chaque pays fait ce qu'il fait le mieux), l'avantage comparatif de Ricardo (même un pays moins bon partout a intérêt à se spécialiser dans ce où il est le MOINS mauvais).
Économies d'échelle Spécialisation Avantage comparatif
Ex. : le marché unique européen — 0 droits de douane entre 27 pays
Pour la protection ★★★★☆
Le protectionnisme
Deux types de barrières : les droits de douane (taxes sur les produits importés — barrières tarifaires) et les quotas (plafonds sur les quantités), normes techniques, subventions aux entreprises nationales (barrières non tarifaires — plus discrètes mais tout aussi efficaces).
Droits de douane Quotas Normes Subventions
Ex. : droits de douane US de 25% sur l'acier chinois
Régulation ★★★☆☆
Réguler et mesurer
L'OMC tente de réguler le commerce mondial (164 pays, principes de non-discrimination). Les accords régionaux comme l'UE créent des zones de libre-échange. Les effets se mesurent avec la balance commerciale (export − import), les IDE (investissements d'un pays dans un autre), les exportations et importations.
OMC Accords régionaux UE Balance commerciale IDE Exportations Importations
Rappels 1ère ★★☆☆☆
Offre, demande et prix
Le libre-échange modifie l'offre et la demande : plus de producteurs = plus d'offre = baisse du prix d'équilibre. L'élasticité-prix mesure si la demande réagit beaucoup ou peu quand le prix change. Le taux de variation sert à commenter les évolutions chiffrées.
Offre et demande Prix d'équilibre Élasticité-prix Taux de variation
Confusions classiques au Bac
⚠️ Libre-échange = DOCTRINE (choix politique) ≠ Mondialisation = PROCESSUS (fait observable)
⚠️ Barrières tarifaires = TAXES (droits de douane) ≠ Non tarifaires = RÈGLES (quotas, normes)
⚠️ IDE entrants = une entreprise étrangère investit CHEZ NOUS ≠ IDE sortants = nos entreprises investissent À L'ÉTRANGER
⚠️ Balance excédentaire = on EXPORTE plus ≠ Balance déficitaire = on IMPORTE plus
Trucs pour retenir
LE = Libre-Échange = Laisser Entrer : on laisse les produits entrer librement
PRO-tectionnisme = PRO-tège : on protège la production nationale
Tarifaire = TARIF = TAXE : c'est une taxe sur les importations
Balance : EX > IM = excédent : si on exporte plus qu'on importe = excédent (positif)
IDE = Investir Dans l'Étranger : une entreprise investit dans un autre pays
Ricardo = Relatif : l'avantage comparatif est RELATIF, pas absolu
